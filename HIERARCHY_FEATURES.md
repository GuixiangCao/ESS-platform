## 🚀 使用方法

### 创建顶级经销商
1. 点击 "新建顶级经销商" 按钮
2. 填写经销商信息
3. 点击 "创建"

### 创建下级经销商
1. 在树形视图或卡片视图中，找到要添加下级的经销商
2. 点击该经销商的 "+" 按钮
3. 填写下级经销商信息（会自动显示上级信息）
4. 点击 "创建"
5. 新创建的下级会自动出现在父级下方，并自动展开父节点

### 编辑经销商
1. 点击经销商的编辑按钮（铅笔图标）
2. 修改信息
3. 点击 "更新"

### 删除经销商
1. 点击经销商的删除按钮（垃圾桶图标）
2. 确认删除
3. **注意**：有下级经销商的无法删除

### 切换视图
- 点击搜索框右侧的 "树形视图" 或 "卡片视图" 按钮

### 搜索经销商
- 在搜索框输入经销商名称或编码
- 结果会在当前视图中高亮显示

## 🎯 核心特性

### 层级字段
```javascript
{
  parentResellerId: ObjectId,    // 父经销商ID
  hierarchyLevel: Number,        // 层级深度（0为顶级）
  path: String,                  // 祖先路径（如: "id1/id2/id3"）
  subResellerCount: Number       // 直接下级数量
}
```

### 设备分配追踪
```javascript
{
  assignmentPath: [ObjectId],    // 从顶级到当前的完整路径
  assignmentHistory: [{          // 分配历史记录
    resellerId: ObjectId,
    assignedAt: Date,
    assignedBy: ObjectId
  }]
}
```

### API 端点
```
GET  /api/resellers/tree/hierarchy        - 获取层级树
GET  /api/resellers/:id/sub-resellers     - 获取子经销商
GET  /api/resellers/:id/ancestors         - 获取祖先路径
PUT  /api/resellers/:id/move              - 移动经销商
POST /api/resellers                       - 创建经销商（支持 parentResellerId）
```
