# 收益率分析功能说明

## 功能概述

收益率分析功能为普通（非AI）电站提供详细的每日收益可视化分析，通过两种不同的视角展示同一组数据，帮助运营团队深入了解电站的收益表现。

**重要**: 此功能仅在普通电站（非AI电站）的每日收益柱状图下方显示。

## 功能位置

1. 进入"电站分析"页面
2. 选择一个普通电站（非AI电站）
3. 查看"每日柱状图"标签页
4. 在每日收益图表下方，会看到"收益率分析"区域

## 两种分析视角

两个tab都显示相同的每日收益柱状图（实际收益柱状 + 预期收益折线 + 可优化金额），但提供不同的分析视角：

### 1. Copilot收益率（默认）

**分析重点**: 关注AI优化带来的潜在收益提升

**图表内容**:
- **绿色柱状图**: 实际收益
- **蓝色折线**: 预期收益
- **浅橙色柱状图**: 可优化金额（如果当前达成率低于AI电站平均达成率）

**适用场景**:
- 评估接入AI系统的价值
- 了解优化潜力空间
- 识别哪些天数可以通过AI优化获得更多收益

### 2. 平均每日收益率

**分析重点**: 关注电站整体收益表现和波动情况

**图表内容**:
- **绿色柱状图**: 实际收益
- **蓝色折线**: 预期收益
- **浅橙色柱状图**: 可优化金额（如果存在）

**适用场景**:
- 分析电站运营稳定性
- 识别异常波动日期
- 评估整体收益表现

## 图表特性

### 零轴坐标系
图表支持负数收益的正确显示：
- **正数收益**: 绿色柱状图，从零轴向上延伸
- **负数收益**: 红色渐变柱状图（带斜纹警告图案），从零轴向下延伸，并有红色边框

### 交互功能
- **鼠标悬停**: 显示详细的收益数据tooltip
  - 日期
  - 预期收益
  - 实际收益
  - 当日达成率
  - AI平均达成率（如果有优化空间）
  - 可优化金额（如果有）
- **负数收益警告**: 当收益为负数时，tooltip会显示"⚠️ 负数收益（支付费用）"

### 图例说明
- **蓝色折线**: 预期收益
- **绿色柱状**: 实际收益
- **浅橙色柱状**: 可优化金额（仅当存在优化空间时显示）

## 使用场景

### 场景 1: 评估AI优化价值（Copilot收益率）
**目的**: 决策是否接入AI系统

**操作步骤**:
1. 选择一个普通电站
2. 默认查看"Copilot收益率"视图
3. 观察图表中的浅橙色柱状图（可优化金额）
4. 查看每天可以通过AI优化获得多少额外收益
5. 评估整体优化潜力

**判断标准**:
- 浅橙色柱状图多且高 → 优化潜力大，建议接入AI
- 浅橙色柱状图少或低 → 优化潜力小，当前运营已较优

### 场景 2: 分析收益波动（平均每日收益率）
**目的**: 了解电站运营稳定性

**操作步骤**:
1. 切换到"平均每日收益率"视图
2. 观察绿色柱状图的高度变化
3. 对比蓝色折线（预期）和绿色柱状（实际）的差距
4. 识别差距较大的日期

**关注点**:
- 绿色柱状图波动大 → 运营不稳定，需排查原因
- 绿色柱状图平稳 → 运营稳定
- 某些天柱状图明显低于折线 → 该日期可能有异常情况

### 场景 3: 识别问题日期
**目的**: 找出收益异常的具体日期

**操作步骤**:
1. 在任一视图中观察图表
2. 找到明显偏低的绿色柱状图
3. 鼠标悬停查看详细数据
4. 记录日期和达成率
5. 进一步查看"损失分析"标签页了解原因

### 场景 4: 对比两种视角
**目的**: 全面了解电站表现

**操作步骤**:
1. 在"Copilot收益率"视图观察优化潜力
2. 切换到"平均每日收益率"视图观察整体表现
3. 综合分析：
   - 如果优化空间大且波动大 → 运营问题+优化机会
   - 如果优化空间大但波动小 → 稳定但有提升空间
   - 如果优化空间小且波动小 → 运营良好
   - 如果优化空间小但波动大 → 需改善运营稳定性

## 数据来源

### AI优化数据
- AI电站平均达成率: `achievementStats.aiStations.averageAchievementRate`
- 来自API: `/api/revenue/achievement-stats`

### 每日收益数据
- 数据源: `stationData.dailyRecords`
- 包含字段:
  - `date`: 日期
  - `expectedRevenue`: 预期收益
  - `actualRevenue`: 实际收益

### 可优化金额计算
```javascript
const currentRate = (actualRevenue / expectedRevenue) * 100;
if (aiAvgRate > currentRate) {
  optimizationAmount = expectedRevenue × ((aiAvgRate - currentRate) / 100);
}
```

## 视觉设计

### 按钮切换
```
┌──────────────┬──────────────────┐
│ Copilot收益率 │  平均每日收益率  │
│   (active)    │                  │
└──────────────┴──────────────────┘
```

- 激活状态：蓝色背景，白色文字
- 未激活状态：透明背景，灰色文字
- 悬停效果：背景变亮

### 图表样式
- **绿色柱状图**: 实际收益（渐变效果）
- **蓝色折线**: 预期收益（带标记点）
- **浅橙色柱状图**: 可优化金额（堆叠在绿色柱上方）
- **红色柱状图**: 负数收益（渐变+斜纹警告+红色边框）
- **灰色虚线**: 零轴参考线

## 注意事项

1. **仅普通电站**: AI电站不显示此功能
2. **数据完整性**: 需要有每日收益数据
3. **优化空间判断**: 只有当普通电站达成率低于AI平均达成率时才显示可优化金额
4. **负数处理**: 完整支持负数收益的显示和计算
5. **性能**: 图表渲染在客户端进行，大数据量时可能需要优化

## 与其他功能的关系

### 与每日柱状图的关系
- 收益率分析使用相同的图表代码
- 两个tab提供不同的分析视角
- 共享零轴坐标系和负数处理逻辑

### 与损失分析的关系
- 收益率分析识别低收益日期
- 损失分析解释收益损失的具体原因
- 两者互补，提供完整的分析链路
