# 收益查看本地化页面 - 完成报告

## 概述

收益查看本地化页面已重新设计为**货币格式化展示工具**,专注于展示全球15种主要货币的格式化效果,而不是业务数据展示。

## 页面功能

### 1. 货币选择区域

**网格式货币卡片**:
- 15种货币以卡片形式展示
- 每张卡片显示:
  - 货币符号(大号显示)
  - 国家/地区名称
  - 货币代码(如 CNY, USD, EUR)
- 点击卡片切换当前货币
- 当前选中货币高亮显示(蓝色边框+背景色)

### 2. 货币格式规范卡片

展示当前选中货币的详细信息:
- **国家/地区**: 如"人民币"、"US Dollar"
- **货币代码**: CNY, USD, EUR 等
- **货币符号**: ¥, $, € 等(大号彩色显示)
- **小数位数**: 0-2位

### 3. 格式化对照表

8个不同数量级的金额展示:

| 数量级 | 示例金额 | 展示内容 |
|--------|---------|----------|
| 百位 | 123.45 | 完整格式 + 缩写格式 |
| 千位 | 1,234.56 | 完整格式 + 缩写格式 |
| 万位 | 12,345.67 | 完整格式 + 缩写格式 |
| 十万位 | 123,456.78 | 完整格式 + 缩写格式 |
| 百万位 | 1,234,567.89 | 完整格式 + 缩写格式 |
| 千万位 | 12,345,678.90 | 完整格式 + 缩写格式 |
| 亿位 | 123,456,789.01 | 完整格式 + 缩写格式 |
| 十亿位 | 1,234,567,890.12 | 完整格式 + 缩写格式 |

**交互功能**:
- 点击任意金额复制到剪贴板
- 复制成功显示绿色对勾图标
- 悬停显示复制图标

### 4. 格式化说明卡片

三栏式布局说明:

**完整格式**:
- 显示完整的金额数值
- 包含千位分隔符
- 保留指定的小数位数
- 适用于需要精确显示的场景

**大额缩写**:
- 自动选择合适的单位缩写
- 保留2位小数确保精度
- 符合各国本地化习惯
- 适用于统计图表和概览

**缩写规则**:
- 中国: 万(1万) · 亿(1亿)
- 美国/英国: K(千) · M(百万) · B(十亿) · T(万亿)
- 日本: 万(1万) · 億(1亿) · 兆(1万亿)
- 印度: K(千) · L(十万) · Cr(千万)

## 技术实现

### 组件结构
```
RevenueView (主组件)
  ├─ 页头
  ├─ 货币选择卡片
  │   └─ 15个货币按钮(网格布局)
  └─ FormatExamples (子组件)
      ├─ 货币信息卡片
      ├─ 格式化对照表
      └─ 使用说明卡片
```

### 状态管理
- `selectedLocale`: 当前选中的货币区域
- `copiedIndex`: 跟踪哪个金额被复制

### 核心功能
- **货币切换**: 点击货币卡片立即更新所有格式化结果
- **复制功能**: 使用 `navigator.clipboard.writeText()`
- **动态格式化**: 根据选中的货币实时格式化8个测试金额

## 用户体验

### 交互流程
1. 进入页面,默认显示人民币(zh-CN)
2. 浏览15种货币卡片,点击任意货币切换
3. 查看货币格式规范信息
4. 在对照表中查看8个数量级的格式化效果
5. 点击金额复制,用于其他地方使用
6. 阅读格式化说明了解使用场景

### 视觉特点
- **货币卡片**: 网格布局,悬停效果,选中高亮
- **对照表**: 清晰的表格结构,等宽字体显示金额
- **复制按钮**: 悬停显示,成功反馈
- **缩写格式**: 蓝色高亮显示,更加醒目

### 响应式设计
- **桌面**: 货币卡片4-5列,表格完整显示
- **平板**: 货币卡片3-4列,表格横向滚动
- **手机**: 货币卡片2列,隐藏原始金额列
