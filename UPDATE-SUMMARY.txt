🌙 暗黑模式工作流程

用户路径:
  1. 首次访问 → 自动检测系统偏好
  2. 手动切换 → 右下角太阳/月亮按钮
  3. 主题保存 → localStorage 持久化
  4. 下次访问 → 自动应用保存的偏好

技术实现:
  • 使用 window.matchMedia('(prefers-color-scheme: dark)')
  • 应用 'dark' 类到 <html> 元素
  • CSS 变量自动切换所有颜色
  • 所有组件自动适配，无需修改

兼容浏览器:
  • Chrome/Edge (最新)
  • Firefox (最新)
  • Safari (最新)
  • 移动浏览器 (iOS Safari, Chrome Mobile)

  暗黑模式 (自动反演):

    • 表面: #1f2937 (表面灰)
    • 文本: #f3f4f6 (浅灰)
    • 强调: #60a5fa (浅蓝)
    • 其他色彩自动适配

排版系统 (7 级)
  • h1: 2rem (32px) - 页面标题
  • h2: 1.5rem (24px) - 分组标题
  • h3: 1.25rem (20px) - 子标题
  • h4: 1.125rem (18px) - 组件标题
  • h5: 1rem (16px) - 标签文本
  • h6: 0.875rem (14px) - 小标签
  • body: 1rem (16px) - 正文

间距标度 (6 级)
  • xs: 4px    - 紧凑
  • sm: 8px    - 小
  • md: 16px   - 默认
  • lg: 24px   - 大
  • xl: 32px   - 超大
  • 2xl: 48px  - 页面级别

💡 最佳实践总结

✅ 使用 CSS 变量代替硬编码颜色
   var(--text-primary) 而不是 #1f2937

✅ 使用间距标度代替任意像素值
   var(--space-md) 而不是 16px

✅ 使用预定义类代替内联样式
   className="btn btn-primary" 而不是 style={{...}}

✅ 检查暗黑模式支持
   所有颜色都应使用 CSS 变量

✅ 测试响应式布局
   在 480px, 768px, 1920px 处测试

✅ 关联标签和输入
   <label htmlFor="id"><input id="id" />
