## ⚡ 快速启动

### 开发环境
```bash
cd ess-platform/frontend
npm install
npm run dev
```

### 查看暗黑模式
1. 打开浏览器开发者工具
2. 右下角点击太阳/月亮图标切换
3. 或按浏览器的深色模式快捷键

### 测试响应式
1. 按 F12 打开开发者工具
2. 点击响应式设计模式 (Ctrl+Shift+M)
3. 选择不同设备尺寸
