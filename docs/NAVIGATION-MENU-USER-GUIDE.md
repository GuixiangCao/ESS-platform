# 导航菜单使用指南 | Navigation Menu User Guide

## 🎯 概述

ESS 平台的 Dashboard 页面已添加一个便捷的下拉菜单，用户可以快速访问三个主要管理模块。

## 📍 菜单位置

菜单按钮位于 Dashboard 页面的 **右上角 header** 区域，具体位置如下：

```
[ESS 平台 Logo]  ...  [用户欢迎文本] [用户头像] [下拉菜单按钮] [退出登录按钮]
```

## 🔍 菜单外观

### 菜单按钮
- **图标:** 向下箭头符号 (↓)
- **位置:** 用户头像和退出登录按钮之间
- **大小:** 40x40 像素
- **颜色:** 与主题一致（亮色/暗色模式自动适应）

### 菜单下拉列表
点击菜单按钮后，会显示一个包含 3 个菜单项的下拉菜单：

```
┌─────────────────────┐
│ 🏢 经销商管理      │
│ 📦 设备管理        │
│ 👥 运维人员        │
└─────────────────────┘
```

## 🚀 使用方法

### 第一步：登录
1. 访问 ESS 平台登录页面
2. 输入用户名和密码
3. 点击登录

### 第二步：进入 Dashboard
登录后会自动跳转到 Dashboard 首页，这是看板管理页面。

### 第三步：打开菜单
在 Dashboard 页面的 header 右侧，点击 **下拉箭头按钮**（在用户头像和退出登录按钮之间）

### 第四步：选择模块
菜单展开后，选择要访问的模块：
- **经销商管理** - 查看和管理经销商账户
- **设备管理** - 查看和管理设备信息
- **运维人员** - 查看和管理员工信息

### 第五步：进入模块
点击选中的菜单项，系统会自动跳转到相应的管理页面。

## 💡 菜单交互

### 展开菜单
```
点击 ↓ 按钮
    ↓
菜单展开，显示 3 个选项
```

### 收起菜单
有两种方式：
1. **点击菜单按钮** - 再次点击 ↓ 按钮关闭菜单
2. **点击菜单外部** - 在菜单外任何地方点击会自动关闭菜单
3. **选择菜单项** - 选择一个选项后菜单自动关闭

### 菜单项悬停效果
- 将鼠标悬停在菜单项上时，项目背景会改变颜色
- 菜单项会显示更明显的色彩

## 📱 响应式设计

### 桌面版 (> 768px)
- 菜单按钮显示清晰
- 完整菜单列表
- 完整的欢迎文本和按钮标签

### 平板版 (768px - 480px)
- 菜单按钮仍然显示
- 隐藏欢迎文本，仅显示头像
- 按钮隐藏文字标签，仅显示图标

### 手机版 (< 480px)
- 菜单按钮仍然可用
- 优化的空间布局
- 适配小屏幕的触摸操作

## 🎨 主题适配

菜单自动适配当前系统主题：

### 亮色模式 ☀️
- 背景色：浅灰色 (#f3f4f6)
- 文本颜色：深灰色 (#1f2937)
- 悬停背景：更浅的灰色
- 图标颜色：蓝色 (#3b82f6)

### 暗色模式 🌙
- 背景色：深灰色 (#2d3748)
- 文本颜色：浅灰色 (#f3f4f6)
- 悬停背景：更深的灰色
- 图标颜色：浅蓝色 (#60a5fa)

## 🔄 菜单项详情

- **图标:** Building2
- **路径:** `/resellers`
- **功能:** 
  - 查看所有经销商列表
  - 创建新经销商
  - 编辑经销商信息
  - 删除经销商
  - 管理权限
  - 搜索和过滤

### 2. 设备管理 📦
- **图标:** Package
- **路径:** `/devices`

  - 添加新设备
  - 编辑设备信息
  - 删除设备
  - 追踪设备状态
  - 分配设备给经销商

### 3. 运维人员 👥
- **图标:** Users
- **路径:** `/staff`
- **功能:**
  - 查看所有运维人员
  - 添加新员工
  - 编辑员工信息
  - 删除员工
  - 管理技能标签
  - 分配工作任务

## ⚡ 快速操作

### 从任何模块返回 Dashboard
- 点击页面 header 的 **ESS 平台** 标题，回到首页

### 快速切换模块
1. 点击菜单按钮
2. 选择目标模块
3. 自动跳转

### 退出登录
- 点击 header 右侧的 **退出登录** 按钮
- 系统会清除会话并返回登录页面

## 🆘 常见问题

### 问题 1：菜单按钮不显示
**解决方案:**
- 检查浏览器是否支持 JavaScript
- 刷新页面
- 清除浏览器缓存
- 尝试不同的浏览器

### 问题 2：点击菜单项后没有反应
**解决方案:**
- 检查网络连接
- 查看浏览器控制台是否有错误
- 等待几秒钟，系统可能在加载数据
- 刷新页面重试

### 问题 3：菜单项文字显示不清楚
**解决方案:**
- 检查系统主题设置
- 调整浏览器字体大小
- 尝试不同的显示缩放比例
- 更新浏览器到最新版本

### 问题 4：菜单在移动设备上不工作
**解决方案:**
- 确保手机浏览器已启用 JavaScript
- 尝试使用其他浏览器
- 检查手机网络连接
- 重新加载页面

## 🔐 权限说明

并非所有用户都能访问所有模块。访问权限由管理员设置：

| 用户角色 | 经销商管理 | 设备管理 | 运维人员 |
|---------|----------|---------|---------|
| 管理员 | ✅ | ✅ | ✅ |
| 经销商 | 仅限自己 | ✅ | ❌ |
| 运维员 | ❌ | ✅ | ✅ |

如果菜单项不可用或灰显，说明您没有权限访问该模块。
