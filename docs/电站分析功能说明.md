# 电站分析功能说明

## 功能概述

电站分析页面位于 **收益管理 > 电站分析**,提供了全面的电站收益数据分析和可视化功能。

## 主要功能

### 1. 电站选择
- 下拉菜单可切换查看不同电站
- 显示电站ID和名称
- 当前共有17个电站数据

### 2. 汇总卡片
显示三个关键指标:
- **总预期收益**: 所有记录的预期收益总和
- **实际年收益**: 所有记录的实际收益总和(重点功能)
- **达成率**: 实际收益/预期收益的百分比
  - ≥90%: 绿色(表现优秀)
  - 70-90%: 黄色(表现良好)
  - <70%: 红色(需要改进)

### 3. 年度收益汇总
- 按年度分组展示收益数据
- 每年显示:
  - 预期收益
  - 实际收益
  - 达成率
  - 数据天数

### 4. 图表可视化
提供两种图表类型切换:

#### 月度统计(柱状图)
- 双柱对比:蓝色(预期)vs 粉色(实际)
- 每月显示达成率
- 鼠标悬停显示详细金额

#### 每日趋势(折线图)
- 双线对比:淡蓝(预期)vs 粉色(实际)
- SVG矢量图,可缩放
- 显示完整时间序列

### 5. 数据表格
- 月度数据详情表格
- 包含月份、预期收益、实际收益、达成率、数据天数
- 可滚动查看所有月份

## 技术实现

### 后端API

#### 1. 获取电站列表
```
GET /api/revenue/stations
```
返回所有电站的汇总信息

#### 2. 获取电站详细数据
```
GET /api/revenue/station/:stationId
```
参数:
- stationId: 电站ID
- startDate (可选): 开始日期
- endDate (可选): 结束日期

返回:
- 汇总统计
- 月度统计
- 每日记录

#### 3. 获取年度汇总
```
GET /api/revenue/station/:stationId/yearly
```
返回按年份分组的收益数据

### 数据模型

**StationRevenue** (station_revenues集合)
```javascript
{
  stationId: Number,        // 电站ID
  stationName: String,      // 电站名称
  currency: String,         // 货币单位(CNY)
  date: Date,              // 日期
  expectedRevenue: Number,  // 预期收益
  actualRevenue: Number,    // 实际收益
  createdAt: Date,         // 创建时间
  updatedAt: Date          // 更新时间
}
```

**索引**:
- `{stationId: 1, date: 1}` (唯一索引)

### 前端组件

**文件**: `frontend/src/pages/StationAnalysis.jsx`

**状态管理**:
- stations: 电站列表
- selectedStation: 当前选中的电站ID
- stationData: 电站详细数据
- yearlyData: 年度汇总数据
- chartType: 图表类型('daily' | 'monthly')

**样式**: `frontend/src/pages/StationAnalysis.css`

## 使用步骤

1. 登录系统
2. 点击左侧菜单 **收益管理**
3. 点击子菜单 **电站分析**
4. 从下拉菜单选择要查看的电站
5. 查看汇总卡片了解整体情况
6. 切换图表类型查看不同视角
7. 滚动查看月度详情表格

## 响应式设计

- **桌面**: 完整功能,三列卡片布局
- **平板**: 单列卡片,图表自适应
- **手机**: 紧凑布局,优化触控体验

## 注意事项

1. 所有API需要认证(Bearer Token)
2. 货币格式使用中文人民币格式(¥)
3. 图表使用渐变色增强视觉效果
4. 达成率根据百分比显示不同颜色
5. 数据实时从数据库读取,无缓存
